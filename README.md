# Webpack для React и SASS

### Запуск
    npm run start
    yarn start

### Билд в продакшн
    npm run build 
    yarn build

### Удаление папки build/
    npm run clear 
    yarn clear

### Структура

    -- node_modules/
    -- build/
        -- img/
        -- js/
            -- bundle.js
        -- index.html
        -- styles.css
    -- src/
        -- // в процессе переделки
    -- webpack.config.js
    -- package.json
